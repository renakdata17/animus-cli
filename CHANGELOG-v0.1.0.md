# Changelog — v0.1.0

## Improvements

- **Code review powered by Claude Opus** — PR reviews and code reviews now use Claude Opus for the strongest reasoning and quality gate enforcement.
- **7-point code review checklist** — The code review directive now includes a comprehensive 7-point checklist covering correctness, safety, performance, error handling, test coverage, documentation, and security.
- **Requirements-aware code review** — Code review now explicitly verifies that task requirements and acceptance criteria are satisfied before approving, closing the loop between planning and implementation quality.
